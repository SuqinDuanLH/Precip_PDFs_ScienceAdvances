# Precip_PDFs_ScienceAdvances

This repository contains the code used to compute precipitation probability density functions (PDFs) and to estimate the extreme-range precipitation scale P_L presented in:
Duan et al. (2026) Extreme-range precipitation probability across global weather systems. Science Advances.
